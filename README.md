# Руководство по созданию навыков (Skills) для Claude — перевод на русский

Перевод оригинального руководства Anthropic: [The Complete Guide to Building Skills for Claude (PDF)](https://resources.anthropic.com/hubfs/The-Complete-Guide-to-Building-Skill-for-Claude.pdf?hsLang=en).

## О чём руководство

Skills позволяют один раз описать рабочий процесс для Claude и затем воспроизводить его стабильно и предсказуемо. В руководстве разобраны создание, тестирование и распространение навыков — как автономных, так и в связке с MCP.

## Что внутри

- Технические требования и структура навыков
- Паттерны для автономных навыков и интеграций с MCP
- Проверенные подходы из реальных сценариев
- Тестирование, итерации и распространение

## Для кого

- Разработчики, которым нужно стабильное выполнение рабочих процессов
- Авторы MCP-коннекторов, встраивающие логику процессов в интеграции
- Продвинутые пользователи, автоматизирующие повторяющиеся задачи
- Команды, которые хотят единообразной работы Claude в организации

На создание первого навыка через skill-creator уходит 15–30 минут — достаточно определить 2–3 процесса, которые хотите автоматизировать.
